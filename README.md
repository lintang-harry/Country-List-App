## Getting Started
1. Clone the repo
   ```sh
   git clone https://github.com/lintang-harry/Country-List-App.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Install NPM react-router-dom
   ```sh
   npm install react-router-dom
   ```
4. Start project
   ```sh
   npm start
   ```
